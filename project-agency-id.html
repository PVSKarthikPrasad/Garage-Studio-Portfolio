AGENCY ID SYSTEM

Fully automated identification of agencies from new installs, prioritizing high-value partners.

100% AUTOMATED (WAS 10HRS/WEEK)
<1min IDENTIFICATION TIME
ZERO MISSED OPPORTUNITIES

THE CHALLENGE

Identifying agencies among new installs was a manual task taking 10+ hours a week. This delay meant missed opportunities for personalized outreach and incorrect ticket routing.

THE SOLUTION

I built an automated system using n8n and Gemini that scans new installs, identifies agencies, scopes their size, and routes them to a dedicated support queue. This freed up time for high-value partnership pitches.
