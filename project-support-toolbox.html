SUPPORT TOOLBOX

A Chrome extension that centralizes diagnostic tools, reducing issue resolution time significantly.

<1min DIAGNOSIS TIME (WAS >15MIN)
1-CLICK COMMON FIXES
100% AGENT ADOPTION

THE CHALLENGE

Support agents had to navigate multiple disparate tools to diagnose issues, leading to slow resolution times (>15 mins for common issues) and context switching fatigue.

THE SOLUTION

I developed a Chrome Extension that brings all necessary diagnostic tools into a single interface. It allows agents to diagnose issues in under a minute and resolve common problems with a single click.

TECH STACK
